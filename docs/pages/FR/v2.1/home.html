Description

Scripts Manager est un outil pour les streameurs, les développeurs, mais aussi tous ceux qui souhaitent automatiser des actions sur leur ordinateur.
Vous pouvez tout de même l'utiliser comme une boite à outil pour ne pas redévelopper un logiciel de A à Z si vous êtes un développeur Java Scripts/NodeJS.

N'hésitez-pas à participer en m'envoyant vos suggestions via le serveur Discord de ma chaîne Twitch: https://discord.gg/BNarUut7Kx.
Dans le cas où vous développez des Addons ou des Scripts, proposez-les au téléchargement depuis votre compte GitHub (je l'ajouterai peut-être dans cette documentation).

Téléchargement

La version est celle présenté dans cette documentation: https://github.com/Arubinu/Scripts-Manager/releases/tag/v2.1.0.
Mais vous pouvez consulter toutes les versions ici: https://github.com/Arubinu/Scripts-Manager/releases.

Vous trouverez le logiciel à télécharger dans la partie Assets (ceci n'est pas un installeur mais le logiciel complet).
Mettez le logiciel à l'endroit où vous voulez le laisser avant de le lancer, évitez les dossiers où il y a une demande d'accès administrateur (ce logiciel ne vous demandera jamais ce genre de droits).

Il est possible que Windows reconnaisse le logiciel comme malveillant, puisque je n'ai pas les moyens de prendre un certificat (bien trop cher).

  • Si Windows bloque le téléchargement, il vous faudra désactiver temporairement votre antivirus.
  • Si Windows supprime le fichier une fois téléchargé, vous pouvez vous rendre dans la partie quarantaine pour autoriser le logiciel (il sera immédiatement remis à sa place).
  • Au lancement du logiciel, Windows vous avertira du potentiel danger, vous pouvez cliquer sur Informations complémentaires puis Exécuter quand même

General

Settings

Tous les réglages de base du logiciel se trouvent ici.

Quelques détails:

  • All folder: vous permet d'utiliser vos propres Addons et Scripts depuis le dossier de votre choix.
  • Run on startup: lance le logiciel au démarrage de l'ordinateur.
  • System Tray on launch: n'affiche pas la fenêtre au lancement (toujours disponible dans la barre d'état système).
  • Import: importe un fichier précédemment exporté (par le bouton de la ligne suivante).
  • Export: enregistre l'intégralité des réglages de Scripts Manager dans un fichier.
  • Reset Scripts Manager: remet le logiciel à zéro, comme au premier lancement.

Addons

Il existe actuellement plusieurs Addons intégrés au logiciel qui donnent accès notamment à OBS Studio, Spotify, et Twitch.
Ceux-ci peuvent être utilisés via des Scripts de votre conception ou Multi Actions.

Notifications

Rend l'affichage des notifications disponible par les différents Scripts existants.
Il est possible de les utiliser depuis le bloc Notification du script Multi Actions.

Il est possible de régler l'opacité, la taille, l'arrondi des coins, ainsi que la position des notifications.

OBS Studio

L'adresse est toujours la même, sauf si vous avez une configuration à deux ordinateurs.
Le mot de passe est à récupérer dans OBS Studio, suivez ces quelques étapes:

  • Cliquez sur Outils situé dans la barre d'outils d'OBS Studio (barre en haut de l'application),
  • Cliquez sur Paramètre du websocket obs, pour ouvrir les paramètres,
  • Cochez Activer le serveur WebSocket si ce n'est déjà fait, sans oublier de cliquer sur Appliquer,
  • Cliquez sur Afficher les informations de connexion puis sur Copier, au niveau du mot de passe,
  • Entrez le mot de passe dans le champ prévu à cet effet sur Scripts Manager.

Spotify

Contrôlez Spotify depuis le script Multi Actions via une commande Twitch ou autre.

Connexion limité

Il vous suffit de cliquer sur Generate pour donner accès à votre compte et générer les tokens.
Cette méthode est très limitée, sauf si vous m'envoyez votre adresse e-mail, afin que je puisse l'ajouter à l'application développeur Spotify.

Connexion étendue

Vous devez créer vous-même une application développeur Spotify, sans oublier de saisir votre propre adresse e-mail pour enlever les limitations.
Cette application vous donne le Client ID ainsi que le Client Secret pour pouvoir générer les tokens associés.

Twitch

Débloque toutes les fonctionnalités du script Multi Actions ainsi que du script Stream Flash.

Cliquez sur le bouton Generate de la ligne Token pour donner accès à votre compte.
Si vous possédez un second compte, il peut servir de Bot plutôt que d'écrire sur le tchat en votre nom.

Dans ce cas cliquez sur le bouton Generate de la ligne Bot Token.
Maintenant il vous suffit de cliquer sur le lien fourni pour qu'il soit copié dans votre presse-papier.
Il vous suffit ensuite:

  • D'ouvrir une fenêtre de navigation privée ou un autre navigateur,
  • De connecter votre compte Bot depuis le site de Twitch,
  • Puis d'entrer le lien copié dans la barre d'adresse du navigateur.

Scripts

Multi Actions

Ce script contient un grand nombre de possibilités d'automatisation séparées en modules comme bon vous semble.

En cliquant sur Toggle Blocks vous aurez accès à tous les blocs de script que vous pourrez utiliser en les faisant glisser dans le module.
Le script étant très complexe, une page lui est dédiée en cliquant ici: Multi Actions.

Stream Flash

Faites clignoter votre écran lorsqu'un message est envoyé dans le tchat Twitch.

La première ligne informe du nombre de flashs survenus pour les différents types d'utilisateurs.

Screen to flash laisse le choix de l'écran sur lequel afficher le flash (souvent celui où OBS Studio est affiché).
Le bouton Reset statistics remet à zéro les compteurs de la première ligne.

Quelques paramètres:

  • Opacity: permet de ne pas trop masquer les informations affichées à l'écran.
  • Duration: est le temps que met le flash pour apparaître et disparaître.
  • Delay: ignore les messages pendant ce temps après qu'un flash s'est produit.
  • Pause: met en pause le système Stream Flash pendant la durée choisie (accessible depuis l'icône Scripts Manager dans la barre d'état système).

Respond to join active le flash lorsqu'un utilisateur arrive sur le stream (même sans parler).
Respond to command active le flash lorsqu'une commande est envoyée dans le tchat Twitch.

Stream Widgets

Affiche des pages Web sur l'un de vos écrans.
Peut être utilisé pour afficher ses alertes, son chat Twitch, ou autre sur un jeu (ne fonctionne pas sur les écrans où un jeu est réglé en plein écran exclusif).

Screen for widgets laisse le choix de l'écran sur lequel afficher les pages web (souvent l'écran principal).

Anchor permet de choisir l'ancrage de base de la page pour les valeurs suivantes.
X, Y, Width, Height peut être fait directement depuis l'icône Scripts Manager dans la barre d'état système (avec l'option Edit Mode).
Pour quitter le mode d'édition, appuyez simplement sur le bouton Échap du clavier.

Indiquez un nom dans le champ Name puis l'adresse de la page à afficher dans le champ URL puis cliquez sur Create. Une fois affiché, le déplacement et le redimensionnement se font à partir de l'option Edit Mode mentionnée précédemment.
